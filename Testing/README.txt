This scripts were used to test accuracy

The test.py file read in the csv file to a dataframe and creates a text file named "testingResults.txt" of actual price vs predicted price from the regression model.

This text file is analyzed in the analyze.py file to determine accuracy based off of given paramaters.

Predictions were within $1000 of actual price for 33% of of claims and within $2000 for 58% of claims.
